# Personal-Website Mateo Posada

This project is a personal portfolio website built using HTML, CSS, Bootstrap, and JavaScript. It includes sections for home, about, work experience, projects, and contact.

How to run the website:
Clone the repository using git clone https://github.com/Posadadev/Personal-Website.git, then open the folder in VS Code and run index.html using Live Server, or open it directly in a browser. The live version can be viewed at https://posadadev.github.io/Personal-Website/

Technologies used: HTML, CSS, JavaScript (jQuery), and Bootstrap.

Libraries and frameworks used: Bootstrap was used for layout, grid system, navbar, and cards (https://getbootstrap.com/docs/5.0/getting-started/introduction/
). jQuery was used for form validation and event handling (https://api.jquery.com/
).

Resources used: Bootstrap documentation for navbar, cards, and grid layout (https://getbootstrap.com/docs/5.3/components/navbar/#responsive-behaviors
, https://getbootstrap.com/docs/5.0/components/card/
, https://getbootstrap.com/docs/5.0/layout/grid/
). MDN documentation was used for CSS styling and background images (https://developer.mozilla.org/en-US/docs/Web/CSS
, https://developer.mozilla.org/en-US/docs/Web/CSS/background-image
). JavaScript and regex validation were referenced from MDN (https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide
, https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_expressions
). The WAVE tool was used to check accessibility (https://wave.webaim.org/
).

Features include a responsive layout, navigation bar, full-screen home section with background image, about section with image, work and skills section, project cards with GitHub links, and a contact form with validation.

Note: The contact form does not send real messages and is only used for front-end validation.
